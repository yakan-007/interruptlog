# InterruptLog Web

InterruptLog は、割り込み前提で作業時間を記録するためのローカル完結型タイムトラッカーです。
この `app/` ディレクトリには、React + Vite で動く Web 版 v1 が入っています。

## できること

- タスクの追加、開始、停止、完了
- 割り込みと休憩の記録
- 押し忘れた過去イベントの追加入力
- 履歴の編集と重複イベントの整理
- 日・週・月・年単位のレポート確認
- JSON バックアップ / 復元
- CSV エクスポート

## 保存の考え方

- データはこのブラウザ内にのみ保存されます
- アカウント、クラウド同期、自動バックアップはありません
- ブラウザのデータ削除や別端末利用では内容は引き継がれません
- 大事なログは設定画面から JSON エクスポートして保管してください

JSON インポートは、現在このブラウザに保存されているデータを貼り付けたバックアップ内容で置き換えます。

`npm run build` を実行すると `dist/` に静的ファイルが出力されます。
そのまま静的ホスティングへ配置できます。
